#  Página de Portfólio Pessoal

Site pessoal desenvolvido como projeto do curso Refatorando.

## 📋 Sobre
Página responsiva que apresenta minha jornada na programação, projetos desenvolvidos e formas de contato.

## 🎨 Características
- Design responsivo (adapta-se a mobile e desktop)
- Navegação fluida
- Seções: Sobre, Projetos, Contato
- Animações CSS suaves

## 🛠 Stack Tecnológica
- HTML5 Semântico
- CSS3 (Variáveis, Media Queries, Flexbox)
- JavaScript (Interatividade básica)

##  Como executar
```bash
git clone https://github.com/mateusdev97443/projeto-portfolio.git
cd projeto-portfolio
# Abra index.html
```
